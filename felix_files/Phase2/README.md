# Phase 2 — Offline Cluster Search

Find Reddit posts that semantically match any of 44 topic clusters built in Phase 1.  
Uses posts from **Nov 2024 – Feb 2025** (different period from Phase 1).

## Two Ways to Run

### Option A — Pool already provided
If `pool/pool_embeddings.npy` and `pool/pool_index.csv` are included:

1. Open `Phase2_Search.ipynb`
2. Skip Section A entirely
3. Run Section B cells top to bottom

No API key needed.

---

### Option B — Generate the pool yourself
If only `assets/` is provided and you have Reddit JSONL files:

1. Drop your JSONL files into `downloads/`
   - Any Reddit data in Arctic Shift format works
   - All 24 subreddits or a subset — no minimum per subreddit
2. Get an OpenAI API key from [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
3. Open `Phase2_Search.ipynb`
4. Paste your key into `OPENAI_API_KEY = ''` in the first cell
5. Run Section A top to bottom (~5 min, ~$0.015 for 5k posts)
6. Then run Section B

Section A saves `pool_embeddings.npy` and `pool_index.csv` to `pool/`.  
Future runs skip Section A automatically.

## Searching

In cell `ph2-pick`, set:
```python
SELECTED_CLUSTER_ID = 9   # Relationship Struggles
```

Run `ph2-pick` → shows top 15 matching posts with similarity scores  
Run `ph2-display` → prints full text of a random top-5 match  
Re-run `ph2-display` → get a different post from the same cluster

## If You Get 0 Matches

- In `ph2-search-setup`, set `SIM_THRESH = 0.50` and re-run from there
- Try a cluster with more posts (higher `n_posts` in the menu)
- Make sure your JSONL files cover the same subreddit types as Phase 1
