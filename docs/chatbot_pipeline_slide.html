Persona Chatbot Pipeline — 兩種生成方式對照

同一篇 Reddit post,比較「經過 CCD」與「不經過 CCD」生出的 persona,在相同對話下的差異。

① 兩種 Pipeline

APost-CCD-Chatbox
Post原始貼文
→
CCD8 段認知個案概念化
→
Persona以 CCD 為角色設定
→
Chatbox對話
先產生 8 段 CCD,再以 CCD 建立 persona。
BDirect Post-Chatbox
Post原始貼文
→
Persona直接以 post 為角色設定
→
Chatbox對話
↑ 跳過 CCD 這一層
直接以原始 post 建立 persona。

② 對照結果 — 同一篇 post(#17 Habit Change「relapsed」)、同樣 4 句訊息、gpt-4o

User 問句A · 經 CCDB · 直接 post
失敗日你腦中想什麼? 「feels like I'm wasting time I can't afford to lose… then the self-criticism kicks in.」 「I feel like I'm failing at everything, which is kinda irrational… I'm working on being kinder to myself.」
覺得能回到正軌嗎? 「I have to, right? Just need to reset… setting some simple goals might help.」 「Yeah… I've done it before, so I just need to start again.」

A 生成依據:CCD(自動化思考/核心信念) B 生成依據:原始 post ⚠️ 危機貼文沿用排除規則
